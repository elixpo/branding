# ref — brand source of truth

Reference material the generators and maintainers treat as canonical.

| File | What it is |
|---|---|

| [`image_models.md`](image_models.md) | The Pollinations image/video model registry (names, pricing, modalities) — the lookup behind model choices in the pipeline. |
